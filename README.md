### 火箭隊培訓營 22梯 七大系統作業

<u>系統列表</u>
- 以下系統均有後台管理，具編輯以上權限帳號可進行CRUD操作
- 另有會員系統(User)，可註冊、修改密碼，具admin層級帳號可管理帳號權限

1. 最新消息 (System01_News)
   - 含分類
   - 單一消息可夾帶多檔、多圖與多連結
2. 相簿系統 (System02_PhotoAlbums)
   - 一個相簿有多張相片
   - 可指定相簿封面 
3. 檔案下載系統 (System03_File)
   - 含分類
   - 檔案含檔名、檔案說明
   - 可點選下載檔案
4. 廣告連結 (System04_Ads)
   - 含分類 (分類可管理是否公開)
   - 廣告內容包含連結、連結名稱、廣告圖片
5. 影音專區 (System05_Video)
   - 含分類 (分類可管理是否公開)
   - 影音為Youtube連結、標題與縮圖
6. 論壇系統 (System06_Forums)
   - 分不同版面(分類)，每個版面可指定admin(版主)
   - 每個版面內有多個文章，文章含標題及內容，註冊會員可回覆文章
   - 各版面計算文章數，文章內計算留言數
7. 商品管理系統 (System07_ProductMgmt)
   - 含分類，分類有兩階層(主分類、子分類)
   - 含品牌(廠牌)
   - 商品紀錄: 分類、品牌、品名、規格、售價，並可上傳多張照片及檔案
